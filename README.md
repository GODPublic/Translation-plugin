# Translation-plugin
This is a modified version of the Google Translate plugin that can translate the web version of discord

这是一个修改版的谷歌全能型翻译插件，可以翻译任何网页，包括网页版discord,免去你每次都要把公告复制粘贴到谷歌翻译查看的麻烦

在开发者模式下加载解压插件即可使用
